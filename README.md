# 🎓 EduNotify

> Smart Student Result Management & Notification System

EduNotify is a full-stack web application developed using **PHP, MySQL, Bootstrap 5, PHPMailer, and FPDF**. It helps educational institutions manage students, subjects, semesters, examination results, automated email notifications, and downloadable PDF marksheets.

## ✨ Features

### 👨‍💼 Admin Module
- Secure Admin Login
- Dashboard with Statistics
- Student Management (CRUD)
- Subject Management (CRUD)
- Semester Management
- Result Management
- Publish/Unpublish Results
- Automatic Email Notifications

### 👨‍🎓 Student Module
- Hall Ticket Search
- Semester-wise Result View
- Result Summary
- PDF Marksheet Download
- Print Result

## 🛠️ Technologies Used

- PHP
- MySQL
- Bootstrap 5
- HTML5
- CSS3
- JavaScript
- PHPMailer

## 🚀 Installation

1. Clone the repository.
2. Import `database/edunotify.sql` into MySQL.
3. Configure `config/database.php`.
4. Configure `config/mail.php` with your Gmail App Password.
5. Start Apache and MySQL using XAMPP.
6. Open:

```
http://localhost/ResultAlertSystem/
```
